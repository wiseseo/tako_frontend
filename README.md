# tako_frontend
<br/>

## 타키쿵 🐙
1. 내 주변 길거리 간식 포장마차를 찾아줍니다.
2. 내가 방문한 길거리 간식 포장마차에 대해 글을 쓸 수 있습니다.
3. 길거리 간식 포장마차, 다른 사람의 글을 찜하기 할 수 있습니다.
<br/>
